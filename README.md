## 🛠️ Core Technology Stack

- **Hosting & Infrastructure**: Google Cloud Run, Docker.
- **Frontend**: React 18, Tailwind CSS, Framer Motion (Animations), Lucide React (Icons), Recharts (Data Viz).
- **Backend**: Node.js, Express, Vertex AI (Gemini 2.5 Pro).
- **Intelligence**: Multi-stage Contextual AI, OCR-based ID Verification, WhatsApp Rumor Scanning.
- **Localization**: Full-site translation via `react-i18next` (EN, HI, PT, ES).
- **Design System**: "Cyber-Fintech" (Slate-950, Cyan Neon, Deep Purple).

## 🛰️ Key Features

### 1. Global Pulse Dashboard
An interactive world map synchronized with real-time election statuses. Highlight borders, track turnout, and dive deep into regional metrics.

### 2. AI-Powered Eligibility Check
A dynamic, country-aware eligibility engine that adapts based on the selected region. Synchronized with national laws (e.g., U.S. Voting Rights Act vs India's Representation of the People Act) and supported by a **Legal AI Expert**.

### 3. Live News Intelligence
Aggregate top news channels (YouTube Live) based on your selected country. Includes a **Gemini 2.5 Pro Pulse Report** which analyzes regional sentiment, stability, and activity matrices.

### 4. The Voter Journey (Roadmap)
A 6-stage interactive roadmap (Registration to Counting) that guides users through official protocols. Includes a floating **Satellite Intel** popover for stage-specific AI advice.

### 5. AI Identity Scanner
Upload any government ID. The system uses **Gemini 2.5 Pro Vision** to parse biometric data, calculate age for 2026, and route you to the nearest polling stations.

## 🌐 Multilingual Synchronicity

VoterQuest is built for the world. Switching languages doesn't just change the buttons—it **re-syncs the AI's brain**. When you switch to Hindi or Portuguese:
1. All UI text translates instantly.
2. The AI's system instructions update to respond in the target language.
3. Regional metrics and radar charts are re-generated in the local script.

## 🛠️ Setup & Deployment

1. **Clone the repository**:
   ```bash
   git clone https://github.com/voterquest/voter-journey.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment**:
   Create a `.env` file and add your Vertex AI and Google Maps keys.
4. **Deploy to Cloud Run**:
   ```bash
   ./deploy.sh
   ```
